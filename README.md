# AI Representation Optimizer

A merchant-facing AI diagnostics platform that helps Shopify stores understand how AI shopping agents perceive their store.

## Tech Stack

- Next.js
- FastAPI
- OpenAI API
- Tailwind CSS
- BeautifulSoup

## Features

- AI Representation Score
- Missing Information Detection
- Recommendation Engine
- AI Perception Summary
- Dashboard Analytics

## Run Frontend

cd frontend
npm install
npm run dev

## Run Backend

cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
